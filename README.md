

# **Blockwork - AI & Blockchain-Powered Freelancing Platform**  

Blockwork is a secure and transparent freelancing platform that utilizes AI and blockchain technology for fair pricing, escrow-based payments, and AI-driven dispute resolution.  

## **Features**  
- **AI Arbitration** – Unbiased dispute resolution.  
- **Blockchain Payments** – Secure, escrow-based transactions.  
- **Dark Mode Support** – Seamless theme toggle.  
- **OAuth Authentication** – Google & Apple login integration.  

## **Tech Stack**  
- **Frontend:** React, TypeScript, Tailwind CSS  
- **State Management:** React Hooks (useState)  
- **Authentication:** Google & Apple OAuth  

## **Getting Started**  

1. **Clone the Repository:**  
   ```sh
   git clone https://github.com/your-username/blockwork.git
   cd blockwork
   ```
2. **Install Dependencies:**  
   ```sh
   npm install
   ```
3. **Run the Project:**  
   ```sh
   npm run dev
   ```
